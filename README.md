### 到这儿

#### 还怕错过站, 不如到这儿

一款基于 RN 的 Android APP, 应用场景是: 坐车时认认真真在看书, 不想突然错过站.

是的, 到这儿帮你解决这个问题, 只需要添加目的地, 设置提醒距离, 到这儿便会在即将到达时提醒你, 再也不用担心坐过站, 再也不用一心两用!

戴上耳机, 享受属于你的行程!
